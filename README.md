# ToolkitSQLite3
## by Markus Jäger
### Version 0.9

---

ToolkitSQLite3 ist eine schlanke PHP-Hilfsklasse für die einfache und sichere Arbeit mit SQLite3.
Sie kapselt typische Datenbankaufgaben wie Tabellen- und Spaltenverwaltung sowie das Upsert von Datensätzen in einer klaren, performanten API.

Features:
- Automatisches Erstellen und Löschen von Tabellen
- Hinzufügen und Entfernen von Spalten (ignore-safe)
- Sicheres UPSERT von Zeilen über _slug
- Strikte Sanitisierung von Tabellen- und Spaltennamen
- Prepared Statements mit korrekter Typzuordnung
- Fokus auf Einfachheit, Geschwindigkeit und SQL-Injection-Sicherheit
